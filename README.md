## 使用方式

### 环境安装

该项目使用CPU运行
**需要有4G以上空闲的内存(注意不是显存)**

**该项目使用QT6.5.3版本**

将ChatGLM.cpp存储库克隆到本地计算机中：

```
git clone --recursive https://github.com/CsBoBoNice/chatglm_cpp_QT
```

如果在克隆存储库时忘记了--recursive标志，请在项目文件夹中运行以下命令：

```
git submodule update --init --recursive
```
